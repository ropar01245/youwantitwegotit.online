# RonJohns Trendy Things: Management & Monetization Guide

## 1. How to Access the Site
*   **Local View (Live now):** Open [http://localhost:3000](http://localhost:3000) in your browser.
*   **Public Site:** [https://ronjohnsronjohns-trendy.vercel.app](https://ronjohnsronjohns-trendy.vercel.app)
    *   *Note: The public site updates only when you push these local changes to GitHub.*

## 2. How to Add New Products
Your "Manual" products (Amazon, ClickBank, etc.) are stored in a simple JSON file.
1.  **Open this file:** `server/manual_products.json`
2.  **Add a new block like this:**
```json
{
  "id": "m19",
  "title": "Your Product Name",
  "price": "$25.00",
  "priceValue": 25.00,
  "image": "https://image-url.com/photo.jpg",
  "category": "Electronics",
  "affiliateUrl": "YOUR_AFFILIATE_LINK",
  "source": "manual"
}
```
3.  **Save the file.** The site will update automatically on refresh.

## 3. How to Manage Categories
To add a new category (e.g., "Fitness"):
1.  **Backend:** In `server/index.js`, add the category name to the `queryMap` with a search term for eBay (e.g., `'Fitness': 'home gym equipment'`).
2.  **Frontend:** In `client/src/App.tsx`, add the category name to the `categories` array: `['All', 'Health', 'Electronics', 'Household', 'E-books', 'Fitness']`.

## 4. How to Monetize & Track Sales
Tracking and payments are handled by the platforms themselves. You don't need to build a tracking system; you just use their dashboards.

### A. Amazon Associates
1.  Log in to [Amazon Associates](https://affiliate-program.amazon.com).
2.  Search for a product and click **"Get Link"**.
3.  Paste that link into the `affiliateUrl` field in `manual_products.json`.
4.  **Tracking:** Check your Amazon Dashboard for "Clicks," "Ordered Items," and "Earnings."

### B. ClickBank
1.  Log in to [ClickBank](https://www.clickbank.com).
2.  Find a product in the Marketplace and click **"Promote"**.
3.  Generate your **HopLink**.
4.  Paste that link into `manual_products.json`.
5.  **Tracking:** Check ClickBank "Analytics" to see exactly which product is selling.

### C. eBay (EPN)
1.  Sign up for the [eBay Partner Network](https://partnernetwork.ebay.com).
2.  Open your `.env` file in the `server/` folder.
3.  Update these three keys:
    *   `EBAY_API_KEY`
    *   `EBAY_API_SECRET`
    *   `EBAY_CAMPAIGN_ID` (This is where your tracking happens!)
4.  **Tracking:** Check the EPN portal to see clicks and sales generated by your site's automated eBay feed.

## 5. Deployment (Going Live)
To make your changes visible to the world:
1.  Open your terminal in the project folder.
2.  Run:
    ```bash
    git add .
    git commit -m "Full store overhaul and monetization update"

    ```
3.  Vercel will detect the push and update your public URL automatically.
